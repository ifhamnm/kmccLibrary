# E-library Management System developed in Spring Boot and Java
- Backend - Spring Boot
- Frontend - Java (Version 8)

# Functions of this Library
It allows user to manage Members, Categories, Books and Issue Books.


# Setup database
create a new database with the name 'kmcclibrarydb' in local MySql.
Update the database name, username, and password in the file application.properties in the resources folder. 


# Login
for login you may use 
- username as 'admin' and password as 'admin'
